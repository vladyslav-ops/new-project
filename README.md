# Git Instructions

## Clone a Repository
```bash
git clone https://github.com/vladyslav-ops/new-project
```
## Checkout to development branch
```bash
git checkout -b development
```

## Check Status
```bash
git status
```

## Stage Changes
```bash
git add <file>

or

git add .
```

## Commit Changes
```bash
git commit -m "commit message"
```

## Push to Remote
```bash
git push origin development
```

## You can also
### Create a Branch
```bash
git checkout -b <branch-name>
```

### Switch Branches
```bash
git checkout <branch-name>
```

### View Commit History
```bash
git log
```

### Merge Branches
```bash
git merge <branch-name>
```
